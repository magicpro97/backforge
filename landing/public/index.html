Supported Providers

One CLI, multiple backend platforms

⚡

Supabase

Open-source Firebase alternative powered by PostgreSQL. Full SQL access, realtime subscriptions, auth, and storage.

🔥

Firebase

Google's app development platform with Firestore NoSQL database, authentication, hosting, and cloud functions.

📦

PocketBase

Open-source backend in a single binary. Embedded SQLite database, realtime subscriptions, auth, and file storage.

🏗️

Appwrite

Open-source backend server for web and mobile apps. Databases, auth, functions, storage, and messaging.
